Repository review · 03

What the source review changed.

The public copy now follows the code and current primary standards, even where repository descriptions or older articles use broader language.

01

Current standards

Media-erasure projects no longer claim blanket NIST compliance: Rev. 2 superseded Rev. 1. seg-test is described as workflow support, not proof of PCI DSS compliance.

NIST SP 800-88 Rev. 2 ↗PCI SSC documents ↗
02

Evidence over labels

SurfaceLens is the only repository in this set with public automated tests. Secure Automated Purge USB has the only GitHub Release; a version tag alone is not described as a release.

03

Operator safety

Destructive media workflows, active recon, credential validation, saved engagement data, and assessment evidence now carry visible handling and authorization boundaries.

04

Provenance & licensing

LinuxSuggester is identified as a legacy upstream snapshot rather than original work. Its attribution and license need remediation; the NoSQLi lab repository also has no declared license.
