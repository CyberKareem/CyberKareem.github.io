24:T4c21,<p>GoodGames, an easy-level Linux OS machine on HackTheBox, the journey begins with a glaring SQL injection flaw, offering us a path to circumvent login authentication and execute union injections for data extraction. Progressing further, we stumble upon a Server-Side Template Injection (SSTI) vulnerability within the profile’s name field, accessible via the admin’s page. Leveraging credentials harvested from the SQL injection, we breach authentication and seize control, ultimately gaining shell access within a Docker container. Exploiting password reuse within the container, we secure SSH access to the host system. Once inside, we exploit Linux’s file permissions and ownership mechanisms, particularly exploiting the mounted home directory, to escalate privileges and attain root shell access.</p><figure><img src="/writing/media/e9ccaa2e185ababc938d3601.webp" alt="HackTheBox “GoodGames” Walkthrough, figure 1" width="1200" height="999" loading="lazy" decoding="async" /></figure><p>Let’s get started! 🚀</p><h2 id="recon-and-enumeration">Recon &amp; Enumeration</h2><p>Let’s use nmap to full scan for open ports and services:</p><figure><img src="/writing/media/e333d153106a22a5bcac333d.webp" alt="HackTheBox “GoodGames” Walkthrough, figure 2" width="836" height="630" loading="lazy" decoding="async" /></figure><p>Let’s check port 80.</p><figure><img src="/writing/media/38c3175c872ec9b1b58cc0eb.webp" alt="HackTheBox “GoodGames” Walkthrough, figure 3" width="766" height="797" loading="lazy" decoding="async" /></figure><p>And we get a login page.</p><figure><img src="/writing/media/8ed03cb0e964e8122bcd0122.webp" alt="HackTheBox “GoodGames” Walkthrough, figure 4" width="715" height="653" loading="lazy" decoding="async" /></figure><p>Let’s capture the login request of a test credentials.</p><figure><img src="/writing/media/a5575439a74344a9ace1ca49.webp" alt="HackTheBox “GoodGames” Walkthrough, figure 5" width="1200" height="453" loading="lazy" decoding="async" /></figure><p>We’re examining SQL injection vulnerability on the login page, a common target.
