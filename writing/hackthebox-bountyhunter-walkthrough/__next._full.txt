Time to inspect the bountylog JavaScript file.</p><figure><img src="/writing/media/d8e32ea329ef196bf1394749.webp" alt="HackTheBox “BountyHunter” Walkthrough, figure 7" width="659" height="442" loading="lazy" decoding="async" /></figure><p>It’s a form allowing value input, with data storage elsewhere. Let's inspect its contents.</p><pre><code data-language="text">development@bountyhunter:~$ cat /opt/skytrain_inc/ticketValidator.py#Skytrain Inc Ticket Validation System 0.1#Do not distribute this file.def load_file(loc):    if loc.endswith(".md"):        return open(loc, 'r')    else:        print("Wrong file type.")        exit()def evaluate(ticketFile):    #Evaluates a ticket to check for ireggularities.    code_line = None    for i,x in enumerate(ticketFile.readlines()):        if i == 0:            if not x.startswith("# Skytrain Inc"):                return False            continue        if i == 1:            if not x.startswith("## Ticket to "):                return False            print(f"Destination: {' '.join(x.strip().split(' ')[3:])}")            continue        if x.startswith("__Ticket Code:__"):            code_line = i+1            continue        if code_line and i == code_line:            if not x.startswith("**"):                return False            ticketCode = x.replace("**", "").split("+")[0]            if int(ticketCode) % 7 == 4:                validationNumber = eval(x.replace("**", ""))                if validationNumber &gt; 100:                    return True                else:                    return False    return Falsedef main():    fileName = input("Please enter the path to the ticket file.\n")    ticket = load_file(fileName)    #DEBUG print(ticket)    result = evaluate(ticket)    if (result):        print("Valid ticket.")    else:        print("Invalid ticket.")    ticket.closemain()</code></pre><p>The <code>eval()</code> function in Python is powerful but potentially dangerous if not used carefully. It allows execution of any valid Python expression or code, making it susceptible to Python Code Injection attacks if malicious input is provided. To safeguard against this, rigorous data validation must precede its use dynamically in applications. For further insights into this vulnerability, refer to <a href="https://vipulchaskar.blogspot.com/2012/10/exploiting-eval-function-in-python.html" rel="nofollow noopener noreferrer" target="_blank">this article</a>.</p><p>We found four invalid tickets “md files” in the python script directory:</p><figure><img src="/writing/media/6e80fc11c40741b1070bbd4b.webp" alt="HackTheBox “BountyHunter” Walkthrough, figure 17" width="1200" height="305" loading="lazy" decoding="async" /></figure><p>The script indicates the ticket is invalid.</p><p>The script’s use of the <code>eval()</code> function without sanitization allows for execution of arbitrary Python expressions, presenting an opportunity for OS command injection. to escalate privileges.</p><p>I gained root shell access by creating a file named “root.md” in the “development” user’s home directory depending on the structure of the invalid tickets found
