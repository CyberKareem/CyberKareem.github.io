Ideally, only individuals designated as server administrators should possess the authority to employ the aforementioned HTTP methods.</p><p>The service/port scan conducted using <code>nmap</code> indicates that port 80 corresponds to a website, and the aspect that stands out is the WebDAV-scan.</p><p>Web Distributed Authoring and Versioning (WebDAV) is an HTTP extension that facilitates website creation and modification using the HTTP protocol. Despite its start in 1996, when the concept seemed promising, caution is warranted due to potential vulnerabilities associated with its usage.</p><p>The webdav scan reveals the presence of methods such as PUT and MOVE. This discovery suggests the possibility of using these methods to upload files.</p><figure><img src="/writing/media/68165f6a0ed94e743a099a39.webp" alt="HackTheBox “Granny” Walkthrough, figure 4" width="942" height="116" loading="lazy" decoding="async"></figure><p>Using davtest tool, I'll determine file upload capabilities and directory creation options.</p><figure><img src="/writing/media/767ed1f4cfdc0aea5edb5fb1.webp" alt="HackTheBox “Granny” Walkthrough, figure 5" width="822" height="782" loading="lazy" decoding="async"></figure><p>It appears that several file types can be uploaded, with the exception of the desired&nbsp;.aspx extension.</p><p>To verify, I will test using the <code>curl</code> command. The <code>-d @text.txt</code> syntax specifies that the file <code>text.txt</code> should be used as the request data.</p><p>Next, an attempt will be made to upload a file with the&nbsp;.aspx extension.</p><figure><img src="/writing/media/38c071de115778edb46b415c.webp" alt="HackTheBox “Granny” Walkthrough, figure 7" width="945" height="606" loading="lazy" decoding="async"></figure><p>As <code>/usr/share/webshells/aspx/cmdasp.aspx</code> in the Kali system.</p><figure><img src="/writing/media/ccfc68901d60885d37847f61.webp" alt="HackTheBox “Granny” Walkthrough, figure 8" width="946"
