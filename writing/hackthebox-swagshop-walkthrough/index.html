This article documents the officially retired Hack The Box SwagShop machine in an authorized environment. Retired status was verified on 2026-07-12. Use the techniques only on systems you own or have explicit permission to test. Commands and platform details may have changed; screenshots are preserved from the original lab session and can contain historical lab identifiers.

SwagShop, an easy-level Linux OS machine on HackTheBox, it presents a Magento eCommerce interface. The process to obtain a shell involves the utilization of two exploits. Firstly, an authentication bypass exploit is employed, granting the ability to create an admin user within the CMS. Subsequently, an authenticated PHP Object Injection is harnessed to achieve Remote Code Execution (RCE) with user privileges. For privilege escalation to the root level, a straightforward exploit of sudo vi is implemented.
