Recon & Enumeration

Use nmapAutomator to full scan for open ports and services:

Visit the target at port 80.

The redirection to the site was not possible as we miss an entry for the target's IP address and domain in the /etc/hosts file.

Now vist the target at swagshop.htb.

Make a directory scanning using the tool Dirsearch.

Review around those results.

The target system is operating on Magento, an open-source e-commerce platform written in PHP. It appears to be using Magento 2014, and we need to confirm the exact version.

Check "searchsploit" for Magento 1.9.

Exploitation:

One of the outcomes is Remote Code Execution. Copy the exploit "37977.py" and examine its content.
